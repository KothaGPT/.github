# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to security@kothagpt.org. Provide:

- A clear description of the issue
- Steps to reproduce
- Impact and suggested mitigation

We will respond within 72 hours with next steps.

## Security Updates

- We use Dependabot to keep dependencies updated
- Security patches are prioritized and released promptly
- Follow responsible disclosure practices

## Supported Versions

Only the latest major version receives security updates.

## Security Best Practices

- Use strong, unique passwords
- Enable 2FA on your GitHub account
- Keep your local environment secure
